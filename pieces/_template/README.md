# Untitled

Notes on the piece — process, materials, references, anything worth keeping.

## Archival rule

This folder is fully self-contained. Drop in everything the piece needs:
images, scripts, vendored libraries, fonts. No external CDNs, no `<script
src="https://…">`. If you reach for a library, copy it into this folder and
pin it forever.

The folder should be portable — zip it, email it, open `index.html`, it
works. That's the test.
